### 📢 소개
---
+ 온라인 게임등 다양한 분야에서 해킹을 막기위해 사용되는 안티치트들의 작동방식을 경험해보기 위함

### 🛠️ 개발환경
---
+ C++
+ Assembly
+ VisualStudio, GitHub
<br/><br/>



### 💡목표
---
+ 기본적인 Anti-Reversing technique 을 이용해 프로그램 보호해보기
+ Assembly 기초 문법 숙지

---
+ CRC Check
+ Detect

  + Software BreakPoint
  + Class Name

  + NtQuerySystemInformation
  + BeingDebugged
